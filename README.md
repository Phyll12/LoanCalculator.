# Fintech Loan Calculator with Risk Assessment and GUI

This project demonstrates a Python-based loan calculator that includes:
- Loan amortization schedules
- Risk assessment based on credit scores and debt-to-income ratio
- Data visualization
- An interactive GUI using Tkinter

## Features
- User-friendly GUI for data input and visualization.
- Clear loan schedules with detailed monthly breakdowns.
- Risk profiling for loan applicants.
